myVinyl - a LAMP/CRUD application

HTML | CSS | PHP | MySQL | JQUERY | RESPONSIVE DESIGN

I collect vinyl L.P's. As my collection grew, I found myself accidentally buying titles I already own. I thought how great it would be to have an app where I could check a title I'm interested in against a database of my collection.


Using LAMP/CRUD methodologies, the myVinyl app solved the problem of a lousy memory and a large record collection.
